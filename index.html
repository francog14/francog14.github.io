1) Estructura y Semántica en HTML5

El contenido web se creó con etiquetas semánticas actuales para mejorar accesibilidad, SEO básico y mantenibilidad del código.

Estructura base

  • Formularios con <label for> correctamente asociados
  • Imágenes con alt descriptivo

2) Hojas de Estilo CSS Externas

Se diseñó una interfaz atractiva y funcional con una hoja de estilos externa, facilitando el mantenimiento.

Buenas prácticas

  • Separación contenido/estilo: <link rel="stylesheet" href="styles.css">
  • Diseño responsivo con Flexbox y Grid
  • Estados interactivos (hover/focus) y accesibles

Fragmento ilustrativo

/* styles.css */
:root{ --primary:#0ea5e9; --surface:#0b1220; }
body{ background:var(--surface); color:#e5e7eb }
.btn{ background:var(--primary); border-radius:10px; padding:.6rem 1rem }
.btn:hover{ filter:brightness(1.05) }
CSS externo = mantenimiento eficiente

3) Validación de Formularios en JavaScript

Se validaron formularios con elementos y atributos apropiados para una experiencia clara.

Elementos y atributos

  • required, type="email", minlength
  • autocomplete, placeholder, pattern
  • Mensajes de error personalizados y no intrusivos

Fragmento ilustrativo

// app.js
const form = document.querySelector('#registro');
form.addEventListener('submit', (e) => {
  const email = form.email.value.trim();
  const pass = form.password.value;
  const okEmail = /\S+@\S+\.\S+/.test(email);
  if(!okEmail || pass.length < 6){
    e.preventDefault();
    alert('Revisa tu correo y que la contraseña tenga al menos 6 caracteres.');
  }
});
Etiquetas asociadas Sugerencias (autocomplete) Feedback inmediato
